# 🥦 Fruits & Vegetables Analyzer

## 📌 Overview
AI-powered web app that detects freshness of fruits & vegetables using Deep Learning.

## 🚀 Features
- Upload image
- Detect fresh/rotten
- Confidence score
- Shelf-life prediction
- Nutritional insights

## 🧠 Tech Stack
- Backend: Python, FastAPI/Flask
- ML: PyTorch (MobileNetV2)
- Frontend: React (Vite)

## ⚙️ How to Run

### Backend
cd backend  
pip install -r requirements.txt  
python main.py  

### Frontend
cd frontend  
npm install  
npm run dev  

## 📊 Model
- Architecture: MobileNetV2
- Dataset: Custom dataset (20k images)
